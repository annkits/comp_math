# Основные возможности
- **Линейная алгебра**: решение СЛАУ прямыми и итерационными методами.
- **Нелинейные системы**: реализация методов типа Ньютона для многомерных задач.
- **Интерполяция и аппроксимация**: инструменты для восстановления зависимостей, включая сплайны и тригонометрическую интерполяцию.
- **Численное интегрирование**: вычисление определенных интегралов с автоматическим контролем точности.
- **Численное исчисление**: поиск корней нелинейных уравнений итерационными методами.

# Структура проекта и модули
1. **Линейные системы** (`gauss.py`, `mpi.py`)
  - **Классический метод Гаусса**: Прямой алгоритм исключения неизвестных.
  - **Метод Гаусса с выбором ведущего элемента**: Алгоритм с повышенной численной устойчивостью.
  - **Метод простых итераций (МПИ)**: Реализация с автоматическим расчетом оценочного числа итераций $N$ для достижения заданной точности.
  - **Метод Зейделя**: Оптимизированный итерационный подход для ускоренной сходимости.

2. **Нелинейные уравнения и системы** (`ne_methods.py`, `mNewton.py`)
  - **Поиск корней**: Методы бисекции (половинного деления), хорд и классический метод Ньютона.
  - **Многомерный метод Ньютона**: Решение систем нелинейных уравнений двумя способами: через СЛАУ и через вычисление обратной матрицы Якоби.
  - **Численный Якобиан**: Автоматическое построение матрицы производных.

3. **Интерполяция и аппроксимация** (`interpolation.py`, `cubic_splines.py`, `trig_interpolation.py`)
  - **Многочлены Лагранжа и Ньютона**: Стандартные формы интерполяции на основе узлов и конечных разностей.
  - **Схема Эйткена**: Итерационный алгоритм интерполяции.
  - **Кубические сплайны**: Построение гладкой интерполяции с расчетом моментов.
  - **Тригонометрическая интерполяция**: Расчет через вещественную и мнимую части для периодических функций.
  - **Визуализация**: Построение графиков (Matplotlib) для сравнения интерполяционных моделей с эталонными функциями.

4. **Численное интегрирование** (`integration.py`)
  - **Методы трапеций и Симпсона**: Классические квадратурные формулы.
  - **Метод двойного пересчета**: Автоматический подбор шага $h$ по правилу Рунге для достижения заданной точности $\varepsilon$.

# Технические особенности
- **Анализ точности**: Включен расчет различных типов погрешностей (остаточная, вычислительная, неустранимая, погрешность метода).
- **Гибкость ввода**: Поддержка чтения данных из текстовых файлов (`.txt`) и консольного ввода.
- **Логирование**: Подробный вывод промежуточных этапов вычислений (таблицы разностей, итерационные шаги, матрицы).
